Live Tracker – Monitoramento de Lives no YouTube

Aplicação web desenvolvida para monitorar, em tempo real, múltiplas lives no YouTube através de uma interface centralizada. Criada para evitar a necessidade de acessar manualmente o YouTube Analytics de cada canal.

A solução permite acompanhar métricas essenciais das transmissões ativas, como visualizações, metas de performance e tempo de exibição. Foi construída com **React e TypeScript** no frontend, e **Node.js + Express** no backend, com persistência de dados em **SQLite3**.

Também desenvolvi uma API personalizada com múltiplas rotas REST, protegida por CORS, que alimenta a aplicação com dados atualizados e performáticos.

Tecnologias utilizadas:
React TypeScript Node.js Express SQLite3 CORS REST API

Status: Ferramenta em uso ativo, trazendo ganho de produtividade ao eliminar etapas manuais de análise em canais com múltiplas lives simultâneas.
