YouTube Competitors Analyser

Desktop application built with Node.js and Puppeteer that helps content creators identify their real competitors on YouTube.

It collects strategic data from competitor channels — such as titles, descriptions, tags, views, and publish dates. Based on that, it generates an optimized list of tags using the most frequent and high-performing ones across the dataset.

The app includes a Windows 11 installer, making it easy to use locally with no technical knowledge required. All results are exported automatically as CSV files.

Technologies used:

Node.js Puppeteer csv-writer JavaScript Electron

Status: Finished and used internally for vertical channel competitor research.
